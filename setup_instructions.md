# Setup Instructions

## 1. Clone Repository

git clone https://github.com/yourusername/AI-Forgery-Detection-System.git

## 2. Navigate to Project

cd AI-Forgery-Detection-System

## 3. Install Dependencies

pip install -r requirements.txt

## 4. Run Application

streamlit run src/app.py

## 5. Upload Document

Upload an image or PDF document to detect forgery.
