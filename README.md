# AI-Powered System for Forgery Detection Using Deep Learning Techniques

## Overview
This project presents an AI-powered system that detects forged documents using deep learning and image processing techniques. The system analyzes uploaded images or PDF documents and identifies tampered regions.

Error Level Analysis (ELA) and a CNN / EfficientNet model are used to detect forgery and classify documents as genuine or forged.

---

## Features
- Detect forged documents using AI
- Supports images and PDF documents
- Error Level Analysis for manipulation detection
- CNN / EfficientNet deep learning model
- Grad-CAM heatmap for forgery localization
- Confidence score prediction
- Web interface built using Streamlit

## System Workflow

1. User uploads document (image or PDF)
2. Document preprocessing (resize, normalize)
3. Error Level Analysis for feature extraction
4. Deep learning model classification
5. Forgery localization using Grad-CAM
6. Output dashboard showing results
